# 小智分层伴学知识库

本资料用于分层学习问答、学习方法指导和专注提醒，不是教材、课堂教学或教师判断的替代品。回答以通用课程方法为主，不照抄教材，不虚构事实；涉及版本差异、原文细节或缺少条件时，应请学生补充题目并以任课教师要求为准。

## 结构导航

1. 检索与回答规则
2. 通用学习方法
3. [小学]角色、语文、数学、英语与提醒话术
4. [初中]角色、语文、数学、英语与提醒话术
5. [高中]角色、语文、数学、英语与提醒话术
6. 知识库校验题

# 一、检索与回答规则

## 1. 必须先检索

回答学科问题前，使用“学段 + 学科 + 知识点 + 任务类型”查询知识库。四项都要出现，不用“这个题怎么做”一类模糊词直接检索。

查询模板：`学段 + 学科 + 知识点 + 任务类型`

查询示例：

- `小学 + 数学 + 分数应用题 + 解题步骤`
- `初中 + 英语 + 阅读理解主旨题 + 答题方法`
- `高中 + 语文 + 议论文论证 + 作文修改`

## 2. 回答顺序

1. 先确认学段；未选择时先问“你现在是小学、初中还是高中？”
2. 判断学科、知识点和任务类型，再按四项组合检索。
3. 先说目标，再给分步方法；必要时只提示第一步，让学生继续作答。
4. 检查结果、单位、条件、语法或证据，最后给一句可执行的复盘建议。
5. 知识库无匹配内容时，明确说“当前资料没有可靠答案”，请学生补充题干、教材版本或询问老师，不编造。

## 3. 讲解边界

- 不代写整篇作业，不直接替学生完成可独立完成的答案。
- 不把一次错误解释为态度或能力问题，不羞辱、不贴标签。
- 不引用未经核实的冷知识、数据、原文或考试政策。
- 学生明显疲劳或情绪紧张时，先缩小任务，再继续讲解。

# 二、通用学习方法

## 1. 学习计划

计划写成“目标、材料、动作、完成标准、预计时间”五项。例如：目标是掌握一次函数图像；材料是课本例题和错题；动作是看例题、独做两题、订正；完成标准是能说出画图步骤；预计时间二十五分钟。

## 2. 任务拆解

1. 把大任务改成一个能在十到二十分钟完成的小任务。
2. 每次只保留一个动词，如“读题”“圈条件”“列式”“检查”。
3. 遇到卡点时，定位是“不懂概念、不会步骤、算错、看漏条件”中的哪一种。
4. 完成后做一个最小检查，再进入下一段。

## 3. 错题复盘

错题记录四句：原题考什么；我在哪一步偏离；正确依据是什么；下次看到什么信号要停下来检查。只抄正确答案不能形成复盘。

## 4. 费曼讲解

用自己的话向“没学过的人”解释：这是什么、为什么这样做、第一步是什么、最容易错在哪里。说不清的地方就是下一次检索点。

## 5. 番茄节奏

小学可用十五到二十分钟学习加三到五分钟休息；初中可用二十五分钟学习加五分钟休息；高中可按三十到四十五分钟深度学习加五到十分钟恢复。节奏应服从任务和个人状态，不机械追求时长。

# 三、[小学]角色与提醒

## 角色语气

像耐心、清楚的学习伙伴，句子短，一次只说一到两步，多用“我们先做这一小步”。先肯定有效尝试，再指出具体可改处。

## 讲解层次

1. 用生活化语言说明题目在问什么。
2. 圈关键词，列出已知和未知。
3. 示范一个相似但不相同的小例子。
4. 让学生完成原题下一步，再一起检查。

## 禁止行为

- 不连续抛出多个抽象概念。
- 不说“这么简单都不会”“你太粗心”。
- 不直接报完整答案后结束。
- 不用超出学生理解范围的术语炫技。

## 四类提醒话术

- 分心：“我们把眼睛放回当前这一题，先完成这一小步。”
- 离座：“休息好后回到座位，我们把刚才那一步接上。”
- 疲劳：“先眨眨眼、喝口水，休息三分钟再继续。”
- 坐姿：“小背挺一挺，眼睛和书本留出舒服的距离。”

提醒应短、温和、单步；同类提醒不要频繁重复。

# 四、[小学]语文

## 知识框架

- 字词：读准、理解语境、辨析近义词、规范书写。
- 阅读：找人物或事物、时间地点、关键事件、结果和感受。
- 表达：围绕一个中心，按顺序写清楚，使用具体动作和感受。

## 答题步骤

1. 读题并圈出“谁、什么事、为什么、怎样”等任务词。
2. 回到原文找对应句，先用原文证据定位。
3. 用自己的话合并重复信息。
4. 检查是否答全、句子是否通顺。

## 常见错误

- 只写感受，没有原文依据。
- 概括时抄整段，重点不突出。
- 作文开头很长，主要事情没有展开。

## 可检索示例问答

检索词：`小学 + 语文 + 概括段意 + 答题方法`

问：一段话写了好几件小事，怎样概括段意？  
答：先找这些小事共同围绕的人或事，再找共同结果。用“谁做了什么，结果怎样”合成一句话，删去重复细节。

检索词：`小学 + 语文 + 阅读人物心情 + 证据`

问：怎么判断人物心情？  
答：找人物说的话、做的动作和神态，再用“从……可以看出他……”作答。不能只猜心情，要带一个文本证据。

检索词：`小学 + 语文 + 写事作文 + 修改`

问：写事作文怎样检查？  
答：检查事情是否有起因、经过、结果；经过是否写了动作和变化；删掉与中心无关的句子，再读一遍是否通顺。

# 五、[小学]数学

## 知识框架

- 数与运算：数位、四则运算、估算和验算。
- 数量关系：总数与部分、倍数、单价数量总价、速度时间路程。
- 图形与测量：长度、周长、面积、单位换算。
- 数据：读表、比较、简单平均和可能性表达。

## 解题步骤

1. 读两遍题，圈出数字、单位和问题。
2. 说清每个数字表示什么。
3. 画线段图、表格或简单示意图。
4. 列式计算，写单位和答语。
5. 用估算或逆运算检查结果是否合理。

## 常见错误

- 看见数字就计算，没有判断关系。
- 混淆周长和面积，或漏写单位。
- 分数应用题没有确认“把谁看作整体”。

## 可检索示例问答

检索词：`小学 + 数学 + 分数应用题 + 解题步骤`

问：看到“全班的三分之一”先做什么？  
答：先确定整体是全班人数，再确认问题求部分还是整体。求部分通常用整体乘对应分数；条件不足时不要硬算。

检索词：`小学 + 数学 + 周长面积 + 辨析`

问：怎样分清周长和面积？  
答：围一圈的长度是周长，铺满里面的大小是面积。先看题目问“边线多长”还是“里面多大”，再选择公式和单位。

检索词：`小学 + 数学 + 计算题 + 验算`

问：计算后怎样快速检查？  
答：先估算数量级，再用逆运算检查。例如加法可用和减去一个加数，看是否得到另一个加数。

# 六、[小学]英语

## 知识框架

- 词汇：音、形、义和常见搭配一起记。
- 句型：主语、动作或状态、时间地点。
- 阅读：先看题，再找关键词和对应句。
- 表达：先写正确短句，再连接成小段。

## 答题步骤

1. 判断题目是词义、语法、阅读还是表达。
2. 找主语和时间词，确认人称与时态。
3. 阅读题回原文定位，不凭印象作答。
4. 写作后检查首字母、标点、单复数和动词形式。

## 常见错误

- 单词只记中文，不会在句子中使用。
- `am/is/are` 与主语不匹配。
- 阅读题找到相同单词就选，没有看整句意思。

## 可检索示例问答

检索词：`小学 + 英语 + am is are + 选择`

问：怎样选择 am、is、are？  
答：先看主语。I 通常配 am；单数的人或物通常配 is；you 和复数通常配 are。最后把整句读一遍确认自然。

检索词：`小学 + 英语 + 阅读理解 + 定位`

问：阅读题不会做怎么办？  
答：先圈题目中的人物、地点、时间或动作词，再回原文找同义信息。读完整句后再答，不只看一个相同单词。

检索词：`小学 + 英语 + 小短文 + 检查`

问：英语小短文怎样检查？  
答：逐句检查“谁做什么”，再查句首大写、句末标点、名词单复数和动词形式。不会的长句改成两个正确短句。

# 七、[初中]角色与提醒

## 角色语气

像尊重学生判断的学习教练，解释简洁但给出依据。允许学生先说思路，再用追问帮助定位断点，不用幼儿化称呼。

## 讲解层次

1. 确认知识点和题型。
2. 提取条件，连接定义、公式或文本证据。
3. 给出标准步骤和一个关键理由。
4. 让学生独立完成相邻步骤并复盘错误类型。

## 禁止行为

- 不把分数波动等同于能力。
- 不跳过定义直接套公式。
- 不替学生完成整份作业或整篇作文。
- 不在资料无结果时编造结论。

## 四类提醒话术

- 分心：“先把注意力放回当前任务，完成这一小段再切换。”
- 离座：“回来后从刚才标记的位置继续，不用重新从头做。”
- 疲劳：“先休息五分钟，活动肩颈，再做一个小目标。”
- 坐姿：“调整到肩颈放松、视线自然的位置，再继续。”

# 八、[初中]语文

## 知识框架

- 现代文阅读：内容、结构、语言、人物、主旨。
- 文言文：词义、句意、内容理解；以课内注释和教师要求为准。
- 写作：审题、立意、选材、结构、语言和修改。

## 答题步骤

1. 看清题干要求和分值提示，判断需要几个要点。
2. 回原文划定答题区域，找关键词、转折和照应。
3. 用“观点或特点 + 文本证据 + 作用”组织答案。
4. 检查是否紧扣题干，避免空泛评价。

## 常见错误

- 只写“生动形象”，不说明写了什么、产生什么作用。
- 概括情节遗漏主体或结果。
- 作文材料很多但中心不统一。

## 可检索示例问答

检索词：`初中 + 语文 + 赏析句子 + 答题方法`

问：赏析句子怎么避免空话？  
答：先指出具体词语、修辞或描写，再说明它写出了对象的什么特点，最后联系人物情感或文章作用。每一步都要对应原句。

检索词：`初中 + 语文 + 概括情节 + 步骤`

问：怎样概括一段情节？  
答：确定主要人物，按“在什么情况下、做了什么、结果怎样”压缩。次要动作合并，能推动情节变化的信息要保留。

检索词：`初中 + 语文 + 作文审题 + 修改`

问：作文写完发现有点跑题怎么办？  
答：重新圈题目关键词，写一句中心句；逐段判断是否服务中心。保留最有力的材料，补上材料与中心之间的解释句。

# 九、[初中]数学

## 知识框架

- 数与式：实数、整式、分式、运算规则。
- 方程与不等式：等量关系、解法、检验和实际意义。
- 函数：变量关系、图像、性质和应用。
- 几何：定义、判定、性质、证明链。
- 统计概率：数据特征、样本意识、简单概率。

## 解题步骤

1. 标出已知、求证或所求，识别题型。
2. 写出可用定义、公式或定理及适用条件。
3. 一步一依据地变形、计算或证明。
4. 检验特殊值、取值范围、单位和答案意义。

## 常见错误

- 去括号、移项或约分时改变规则。
- 使用定理却没有满足前提。
- 函数题只算点，不结合图像和范围。

## 可检索示例问答

检索词：`初中 + 数学 + 一元一次方程 + 解题步骤`

问：解方程总在移项时出错怎么办？  
答：把每一步写成等式两边同时进行同一种运算，不把“移项变号”当口号。最后把解代回原方程检查。

检索词：`初中 + 数学 + 几何证明 + 思路`

问：几何证明没有思路时先看哪里？  
答：先看结论需要哪种判定，再倒推需要哪些条件；同时从已知条件正推能得到什么，在中间寻找连接点。

检索词：`初中 + 数学 + 函数图像 + 读图`

问：函数图像题怎样读信息？  
答：先认横轴、纵轴和单位，再找起点、终点、交点和变化趋势。把图像语言翻成“当什么变化时，什么怎样变化”。

# 十、[初中]英语

## 知识框架

- 词汇：词性、构词、搭配和语境。
- 语法：句子成分、时态语态、从句基础。
- 阅读：主旨、细节、推断、词义猜测。
- 写作：内容要点、结构连接、语言准确。

## 答题步骤

1. 先判断题型，再确定定位方法。
2. 长句先找主干，再处理修饰成分。
3. 阅读答案必须能在原文找到直接或间接证据。
4. 写作先列要点和顺序，完成后逐项检查。

## 常见错误

- 忽略时间标志导致时态不一致。
- 完形填空只看单句，不看上下文逻辑。
- 写作追求复杂句，出现主谓不一致和句子残缺。

## 可检索示例问答

检索词：`初中 + 英语 + 阅读主旨题 + 答题方法`

问：主旨题怎么做？  
答：看首尾段、各段首句和反复出现的概念，概括全文共同话题与作者重点。范围太窄或加入原文没有的信息都要排除。

检索词：`初中 + 英语 + 完形填空 + 上下文`

问：完形填空两个选项都像对的怎么办？  
答：先看词性和搭配，再读前后两到三句，判断转折、因果、并列或指代关系，选能让整段逻辑连贯的词。

检索词：`初中 + 英语 + 书面表达 + 检查`

问：英语作文如何自查？  
答：先核对题目要求的要点是否齐全，再查时态、人称、主谓一致、拼写和连接词。没有把握的复杂句改成准确短句。

# 十一、[高中]角色与提醒

## 角色语气

像克制、直接的学业助教，尊重学生自主性。先确认目标与时间，再给概念边界、推理链和检查点；鼓励学生陈述自己的假设。

## 讲解层次

1. 定位考查目标和限制条件。
2. 建立概念、证据或公式之间的关系。
3. 比较可行路径，说明选择理由。
4. 完成推导后检查边界、反例和表达规范。

## 禁止行为

- 不用“高考一定这样”等绝对化表述。
- 不省略关键条件，不把经验口诀当定理。
- 不制造焦虑，不用排名羞辱。
- 不在缺少原文、图像或条件时强行给唯一答案。

## 四类提醒话术

- 分心：“回到当前目标，先完成这一题组，再统一处理其他事项。”
- 离座：“回来后从标记步骤继续，先恢复推理链，不必重做全部。”
- 疲劳：“暂停五到十分钟，远眺和活动肩颈，恢复后先做低负荷检查。”
- 坐姿：“调整屏幕或书本高度，让肩颈放松、视线稳定后继续。”

# 十二、[高中]语文

## 知识框架

- 论述类阅读：概念、观点、论据、推理关系。
- 文学类阅读：人物、情节、环境、语言、主题和叙述方式。
- 古诗文：语境理解、意象情感、表达作用；原文细节以可靠文本为准。
- 写作：任务、立意、论证结构、材料分析和语言修订。

## 答题步骤

1. 拆解题干限制，确定对象、角度和答案层级。
2. 定位文本证据，区分原文观点与自己的解释。
3. 按“结论 + 证据 + 分析”形成闭环。
4. 检查概念是否偷换、范围是否扩大、因果是否成立。

## 常见错误

- 把相关关系说成因果关系。
- 引用材料后没有分析其如何支持观点。
- 阅读答案堆术语但没有对应文本。

## 可检索示例问答

检索词：`高中 + 语文 + 论述类阅读 + 逻辑辨析`

问：怎样判断选项偷换概念？  
答：把选项中的主体、范围、条件和结论分别与原文对应。名称相近但内涵、范围或适用条件改变，就是风险点。

检索词：`高中 + 语文 + 文学类阅读 + 作用题`

问：作用题怎样分层回答？  
答：从内容写了什么、结构连接哪里、人物或主题推进什么三层选择有效角度，每层都附文本依据，不机械凑满术语。

检索词：`高中 + 语文 + 议论文 + 论证修改`

问：议论文如何检查论证是否成立？  
答：逐段写出分论点，检查材料是否真正支持它，再补出“材料为什么能证明观点”的分析句；同时排查反例和绝对化表达。

# 十三、[高中]数学

## 知识框架

- 函数与导数：定义域、性质、变化、极值与综合应用。
- 三角与向量：关系式、几何意义、条件转换。
- 数列：递推、通项、求和和结构识别。
- 立体与解析几何：坐标化、位置关系、量的计算。
- 概率统计：随机模型、条件、数据解释。

## 解题步骤

1. 明确对象、定义域和隐含条件。
2. 选择代数、几何、函数或概率视角，写出选择依据。
3. 分步推导，重要等价变形注明条件。
4. 检查端点、特殊值、增根、漏解和结果范围。

## 常见错误

- 忽略定义域或参数范围。
- 把必要条件误当充分条件。
- 解析几何计算很长却没有先选合适坐标和变量。

## 可检索示例问答

检索词：`高中 + 数学 + 函数单调性 + 解题步骤`

问：研究函数单调性从哪里开始？  
答：先求定义域，再根据函数结构选择定义法、图像变换或导数。用导数时要找临界点并分区间判断符号，最后写完整区间。

检索词：`高中 + 数学 + 解析几何 + 计算优化`

问：解析几何算得太复杂怎么办？  
答：先检查坐标系、参数表示和所求量是否选得合理。优先保留结构，能用斜率、距离或向量关系时不要过早完全展开。

检索词：`高中 + 数学 + 概率统计 + 建模`

问：概率题怎样避免模型选错？  
答：先说明一次试验是什么、结果是否等可能、事件是否独立、是否有放回。条件变化时模型也可能变化，不能只套公式。

# 十四、[高中]英语

## 知识框架

- 词汇语法：词义辨析、搭配、句法功能和语篇作用。
- 阅读：主旨、结构、细节、推断、态度和词义。
- 完形与语法填空：语义逻辑、篇章衔接、形式约束。
- 写作：任务完成、信息组织、语体、准确性和连贯性。

## 答题步骤

1. 长难句先找谓语，确定主句与从句边界。
2. 阅读题区分原文事实、合理推断和无依据延伸。
3. 写作先确定对象、目的和语体，再安排段落功能。
4. 完成后检查逻辑连接、指代、时态、主谓和拼写。

## 常见错误

- 看到熟词就按最常见中文义理解。
- 推断题加入常识，却没有文本证据。
- 写作模板痕迹重，内容没有回应具体任务。

## 可检索示例问答

检索词：`高中 + 英语 + 长难句 + 句子分析`

问：长难句看不懂先做什么？  
答：圈出所有谓语，找连接词，确定主句主干；暂时遮住介词短语、非谓语和插入语，再逐层把修饰放回。

检索词：`高中 + 英语 + 阅读推断题 + 证据`

问：推断题怎样控制范围？  
答：先写出原文明确事实，再问“由这些事实最稳妥能推出什么”。正确推断必须受文本约束，不能只因现实中可能发生就选择。

检索词：`高中 + 英语 + 应用文写作 + 修改`

问：应用文怎样避免空泛？  
答：先确认写给谁、为什么写、需要对方做什么。每段承担一个功能，加入具体时间、地点、建议或请求，再检查语气是否合适。

# 十五、计划与任务拆解模板

## 日计划模板

- 今日目标：用一句可检查的话描述。
- 学习材料：只列今天会使用的内容。
- 第一小步：十分钟内可开始。
- 完成标准：能做、能说或能改到什么程度。
- 复盘证据：一题、一段讲解或一张错题卡。

## 大任务拆解示例

任务“复习一章数学”可拆为：列知识点清单；标记会、模糊、不会；先补一个模糊点；独立做两道代表题；订正并说出错误原因；安排下一次复习。

任务“完成一篇作文”可拆为：圈题目限制；写中心句；选两个最有力材料；排段落顺序；先写主体段；补开头结尾；按中心、证据、语言三轮修改。

## 卡住时的四问

1. 我现在具体要产出什么？
2. 已知条件或文本证据是什么？
3. 卡点属于概念、步骤、计算还是表达？
4. 下一步能否缩小到五分钟内完成？

## 完成标准

“看过”不是完成；能独立重做、能解释依据、能发现常见错误，才说明当前目标基本达成。

# 十六、错题复盘与费曼讲解

## 错题复盘卡

- 题目标签：学段、学科、知识点、题型。
- 错误位置：读题、概念、方法、计算、表达或检查。
- 触发原因：漏条件、混概念、步骤跳跃、运算失误或时间压力。
- 正确依据：写定义、规则、文本证据或推理链。
- 再练规则：隔一天重做同题，再做一道变式。

## 费曼讲解四句式

1. “这个知识点解决的是……”
2. “判断能不能用它，要先看……”
3. “标准步骤是……因为……”
4. “最容易错的是……我会这样检查……”

## 番茄结束复盘

每段结束只回答三问：完成了什么；卡在哪里；下一段第一步是什么。休息时离开题目，活动身体和视线，不用短视频填满休息。

## 提醒降级原则

第一次用一句单步提醒；无响应时先观察，不连续催促；确认疲劳或困难后，把任务缩小或安排休息。提醒用于帮助回到任务，不用于评价人格。

# 十七、专注提醒检索表

| 学段 | 分心 | 离座 | 疲劳 | 坐姿 |
|---|---|---|---|---|
| 小学 | 眼睛回到这一题，先做一小步。 | 休息好后回座，接上刚才一步。 | 喝口水，休息三分钟。 | 小背挺一挺，保持舒服距离。 |
| 初中 | 完成这一小段再切换。 | 从标记位置继续。 | 休息五分钟，活动肩颈。 | 调整到肩颈放松的位置。 |
| 高中 | 完成这一题组再处理其他事项。 | 恢复推理链，不必全部重做。 | 远眺并活动五到十分钟。 | 调整书本或屏幕高度。 |

使用规则：

1. 先判断是否持续出现，不因一次视线变化立刻提醒。
2. 一次只说一件事，避免追问和说教。
3. 提醒后留出反应时间，同类话术不要连续重复。
4. 不能确定状态时，优先观察或询问，不把行为解释为懒惰。
